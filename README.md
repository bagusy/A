# Login Form

This is a minimal HTML/CSS/JS login form demo.

Files:

- [index.html](index.html) — form markup
- [styles.css](styles.css) — styling
- [script.js](script.js) — client-side validation and simulated submit

How to run:

Open `index.html` in your browser.

Next steps you might want:

- Hook the form to a backend `/login` endpoint using `fetch`.
- Add stronger validation and error handling.
- Add tests or integrate with a framework.
